Термины(Да-да, тут даже есть свои термины):
	Контролёр - некая сущность, на подобии игрока.
		Контролёр владеет и манипулирует всеми враждебными юнитами к юнитам игрока.
		Контролёр и игрок - это две враждующие стороны.
		Контролёр не будет как таковой реализован в игре(зависит от разработки).
		Это лишь слово для упрощения дальнейшего описания всего происходящего в игре.

	Режим управления - разновидность инструментов для игрока,
		которые позволяют каким-либо образом взаимодействовать с игрой.
		Благодаря режимам управления игрок может перемещать камеру или отдавать приказы.

[Что изночально есть на карте]
	Карта состоит из 5 областей(4 острова и 1 данж).
	В каждой области есть стартовый набор юнитов для контролёра.

[Старт]
	Игрок со стартовым набором ресурсов и юнитов появляется на случайном острове.
	После появления игрока, контролёр начинает развиваться.

[Победа]
	Победа для одной из враждующий сторон наступает тогда,
	когда противоположная сторона теряет всех своих юнитов типа <Постройка>.

[Классификация юнитов]
	Все юниты делятся на классы(<Постройка> и т.д.).
	Класс делится на типы(класс <Мобильный>, тип <Строитель>)
	
	Классы юнитов:
		<Мобильный>
		<Постройка>
		<Вспомогательный>

	Типы юнитов класса <Мобильный>:
		|Строитель|
		|Воин|
		|Собиратель|

	Типы юнитов класса <Постройка>
		*Неизвестно* 
		Макс, это к тебе.

	Типы юнитов класса <Вспомогательный>
		|Ресурс|
		|Фундамент|

[Общая характеристика всех юнитов]
	С юнитом всегда можно каким-либо образом взаимодействовать.
	Также и сам юнит может взаимодействовать с другими юнитами.
	Все юниты должны находится в одной плоскости.

[Характеристика юнитов классов <Мобильный> и <Постройка>]
	Юнит имеет здоровье.
	Если оно заканчивается, то юнит погибает.
	Восстановить здоровье никак нельзя.
	Юнита нельзя воскресить.

[Класс юнитов <Мобильный>]
	Только такой юнит способен перемещаться в пространстве.
	У юнита есть своя скорость перемещения.
	Юнита нельзя замедлить или ускорить.
	Перемещени происходит по кординатам X и Z.

[Тип юнитов |Строитель|]
	Входит в класс <Мобильный>
	Юнит может создавать юнитов класса <Постройка>, взаимодействуя с юнитом типа |Фундамент|.

[Тип юнитов |Воин|]
	Входит в класс <Мобильный>
	Юнит способен приносить вред враждебному юниту.

[Тип юнитов |Собиратель|]
	Входит в класс <Мобильный>
	Юнит может извлекать ресурс из юнитов типа |Ресурс|.

[Класс юнитов <Постройка>]
	Такого юнита можно создать только взаимодействуя с юнитом типа |Фундамент|.

[Класс юнитов <Вспомогательный>]
	Такой юнит никогда не будет принадлежать ни игроку, ни контролёру.
	Не является противником для игрока и контролёра.

[Тип юнитов |Ресурс|]
	Входит в класс <Вспомогательный>
	Юнит хранит в себе определённый ресурс, который можно каким-либо образом извлечь.

[Тип юнитов |Фундамент|]
	Входит в класс <Вспомогательный>
	Только благодаря этому юниту можно создать юнита типа |Постройка|

[Логика развития контролёра]
	*Нельзя описать, не зная типов юнитов класса <Постройка>

[Стартовый набор юнитов и ресурсов]
	*Нельзя описать, не зная типов юнитов класса <Постройка>
	
[Режимы управления]
	/Осмотр/,
	/Выбор/,
	/Командование/

[Режим управления /Осмотр/]
	Режим, позволяющий игроку перемещать камеру для просмотра местности вокруг.
	Можно отдалять камеру или приближать.

[Режим управления /Выбор/]
	Позволяет выбрать юнита или группу юнитов.
	Игрок может взаимодействовать с его юнитами только в том случае,
	если они были выбраны.

[Режим управления /Командование/]
	Даёт приказ выбранным юнита. Приказ зависит от того, на что указал игрок.
